## Quick Start (5 Steps)

### 1️⃣ Install WordPress & WooCommerce
- Install WordPress on your hosting (Hostinger, Bluehost, etc.)
- Install **WooCommerce** plugin
- Set currency to **Indian Rupee (INR)**

### 2️⃣ Install WordPress Importer
- Go to Tools → Import
- Install **WordPress** importer plugin

### 3️⃣ Import Site Structure
- Upload **aarzah-wordpress-export.xml**
- This imports all pages, categories, and site structure

### 4️⃣ Import Products
- Go to WooCommerce → Products → Import
- Upload **aarzah-woocommerce-products.csv**
- This imports 6 products with pricing and descriptions

### 5️⃣ Import Coupons
- Go to WooCommerce → Coupons → Import
- Upload **aarzah-woocommerce-coupons.csv**
- This imports 5 active discount codes

## Troubleshooting

### XML Import Fails?
✅ **FIXED** - The XML now has proper WXR version 1.2 format
- File is UTF-8 encoded
- Contains proper WordPress namespaces
- Fully validated WordPress export structure

### CSV Import Issues?
- Ensure CSV delimiter is set to **Comma (,)**
- Check that headers match expected format
- Try importing one file at a time

### Images Not Showing?
- Upload to WordPress Media library via Media → Add New
- Or upload via FTP to `/wp-content/uploads/`
- Update product URLs in WooCommerce
